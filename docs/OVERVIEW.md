# Aeroport Management System (Frontend Only)
## Project Overview

Aeroport Management System is a modern aviation management dashboard interface designed to manage airport operations through multiple role-based dashboards.

This project focuses only on frontend UI development using React and Tailwind CSS.

The system provides structured dashboards for different airport roles such as administration, pilots, passengers, operations, maintenance, and ticket booking.

Each dashboard provides its own menus, UI components, tables, forms, and analytics panels.

The interface follows modern aviation design standards with responsive layouts suitable for desktop, tablet, and mobile devices.

## Technology Stack

- **Frontend Framework**: React
- **Styling**: Tailwind CSS
- **Icons**: Heroicons / Lucide Icons
- **Charts**: Recharts or Chart.js
- **Routing**: React Router

## Project Goals

- Create a clean and modern aviation dashboard interface.
- Provide six independent dashboards.
- Ensure each dashboard contains exactly eight menu pages.
- Build reusable UI components.
- Ensure the entire UI is fully responsive.

## Target Users

- Airport Administration Staff
- Pilots
- Passengers
- Flight Operations Managers
- Maintenance Teams
- Ticketing & Booking Teams

## Core UI Features

- Sidebar Navigation
- Top Navigation Bar
- Dashboard Cards
- Data Tables
- Forms
- Notification Panel
- Profile Dropdown
- Responsive Layout System

## Responsiveness

- **Mobile**: 320px – 480px
- **Tablet**: 481px – 768px
- **Desktop**: 769px and above

## Output

A clean frontend dashboard system with structured dashboards and independent menu pages.
